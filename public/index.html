Sick of timetables where you have class at 8am? Want to get home before 5pm? Are there some days you need some extra time? If this sounds like you, you're in the right place. Corsair is your course planning tool.

Lay down some rules about when you want to be at uni, pick some courses, and we'll find the best combination to fit you.





Ready to make life easier?

Why use Corsair?

Getting a great time table is often hard, if not impossible. Flipping through Course catalogues to see which you can fit into a timetable, re-shuffling, finding out you can't do a certain time - sound familiar? With Corsair you select the courses which interest you, prioritise them, add some time constraints and you are given a timetable which best suits you!

Give it a go! Get started now.

What is Corsair?

Developed using Ruby on Rails for the Wellington Summer of Tech Hackfest 2010 this tool scrapes a course list from Victoria University and provides an interface for showing which courses and course streams best fit a given set of rules.
